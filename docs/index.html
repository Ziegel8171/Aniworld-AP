How It Works

  1. Navigate to aniworld.to and select an anime episode
  2. Extension detects the anime, season, and episode number
  3. Fetches skip times from the AniSkip API
  4. Injects skip controls into the video player
  5. Automatically skips segments (when enabled)
  6. Auto-plays next episode (when enabled)
